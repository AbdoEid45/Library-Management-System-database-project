# Library Management System Database Project

This repository contains the database objects for a Library Management System, including:

- Tables and derived attributes
- Data population scripts
- Views, functions, stored procedures, and triggers
- Sample queries

See `Summary of LMS Database Objects.pdf` for details.

## Setup
- Database: SQL Server (or compatible).
- Run scripts in order: 1___CREATE_TABLES_WITH_DERIVED_ATTRIBUTES.sql → ... → _________Final_TriggersAndProceders_8.sql.
